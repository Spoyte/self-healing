# Self Healing Infra

Self-Healing Infrastructure System

## Features

- Comprehensive test suite
- Python-based implementation
- CLI interface
- Easy to use
- Lightweight and fast

## Installation

```bash
cd self-healing-infra && pip install -r requirements.txt  # if Python
```

## Usage

```bash
# Run the script
python self_healing.py

# Or make executable and run
chmod +x self_healing.py
./self_healing.py
```



## Configuration

Configuration can be done via JSON/YAML files in the project root.
